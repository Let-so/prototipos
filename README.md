# AlDía - Obra de Arte Interactiva

> Tu bienestar, tu información, tu decisión. Prototipo persona-céntrico.

## 🌐 Acceso Directo

**Visita la aplicación en vivo:**
🔗 **[https://Let-so.github.io/prototipos/](https://Let-so.github.io/prototipos/)**

## 📋 Descripción

AlDía es una plataforma interactiva diseñada para ofrecer información sobre servicios de salud y bienestar de forma accesible y centrada en la persona.

## 📁 Contenido del Repositorio

- `docs/index.html` - Aplicación principal (GitHub Pages)
- `AlDia - App.html` - Archivo fuente
- `uploads/` - Proyectos y prototipos adicionales
  - `aldia-completo-actualizado/` - Versión completa del proyecto
  - `aldia-organic-calendar-explorar/` - Calendario interactivo
- `screenshots/` - Capturas de pantalla
- `scraps/` - Archivos de referencia

## 🚀 Despliegue

Este proyecto está desplegado automáticamente en GitHub Pages desde la rama `main`, carpeta `/docs/`.

Para cualquier cambio, simplemente haz push a la rama main y los cambios se reflejarán en el sitio en vivo.
